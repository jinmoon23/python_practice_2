# Python Coding Test Setup

## Create and use the virtual environment

```bash
source .venv/bin/activate
```

## Run a Python file

```bash
python main.py
```

## Example with input

```bash
python main.py < input.txt
```

## Notes

- The project uses a local virtual environment at `.venv`.
- For coding tests, the starter file is `main.py`.
- If you open this folder in a VS Code compatible editor, the interpreter is preconfigured.
